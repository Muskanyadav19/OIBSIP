# Calculator

## Project Title
Calculator using HTML, CSS & JavaScript

## Project Description
This is a simple and responsive Calculator web application developed using HTML, CSS, and JavaScript. The calculator performs basic arithmetic operations with a clean and user-friendly interface. This project helped me improve my understanding of DOM manipulation, JavaScript functions, and responsive web design.

## Features
- Perform Addition, Subtraction, Multiplication, and Division
- Clear (AC) button to reset the display
- Delete (DEL) button to remove the last entered digit
- Decimal number support
- Responsive and modern user interface
- Easy-to-use layout

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure
```
Calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
4. Start using the calculator.
